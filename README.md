# rest_json_api

RESTful web application to do CRUD operations on a music library. 

Web Stack:
Database: SQLite
Server: Express.js (Node.js)
HTML template engine: Jade
JQuery


YOu will need to install:

Node.js
npm
sqlite

then run
> npm install 





URL: locahost:3000
